# icc2014
# 問題描述
完成一個系統，其訊號界面如圖一所示，內容包含一序列傳輸介面處理電路(Serial Transmitter 
Interface, STI)及一資料排列控制電路(Data Arrange Controller, DAC) (如圖二)。STI 電路動作為從並
列埠進行資料輸入處理後由序列埠將處理完成之資料以序列輸出。DAC 電路之功能為將經 STI 電
路處理完成後之序列資料進行排列後分別寫入指定記憶體
